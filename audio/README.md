# Audio Files Setup Guide

2. `waiting_unloading.mp3` - "ان لوڈنگ کے قریب انتظار کر رہے ہیں"
3. `unloaded_inside.mp3` - "خالی ہوا، اندر ہے"
4. `empty_left.mp3` - "خالی ہے، چلے گئے"
5. `empty_returning.mp3` - "خالی ہے، واپس جا رہے ہیں"

## How to Record Audio Files

### Option 1: Using Your Phone
1. Open Voice Recorder app on your phone
2. Record yourself saying each phrase clearly in Urdu
3. Save each recording with the filename above
4. Transfer files to your computer
5. Upload to GitHub in the `audio/` folder

### Option 2: Using Computer
1. Use Audacity (free) or any audio recorder
2. Record each phrase
3. Export as MP3 format
4. Save with correct filename
5. Upload to GitHub

### Option 3: Online Text-to-Speech
1. Use Google Translate text-to-speech
2. Type the Urdu text
3. Click the speaker icon to hear it
4. Use screen recording software to capture the audio
5. Save as MP3 with correct filename

## Audio File Requirements
- Format: MP3
- Quality: 128kbps or higher
- Duration: 2-4 seconds per file
- Language: Urdu (clear pronunciation)
- Volume: Loud and clear

## Upload Instructions

1. Go to your GitHub repository
2. Navigate to the `audio/` folder
3. Click "Add file" → "Upload files"
4. Drag and drop all MP3 files
5. Click "Commit changes"

## Testing

After uploading, open the form and tap the 🔊 button on any status. You should hear the Urdu audio.

## Troubleshooting

If audio doesn't play:
- Check that files are in the `audio/` folder
- Verify filenames match exactly (case-sensitive)
- Make sure files are MP3 format
- Check browser console for errors
- Try refreshing the page
